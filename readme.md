# NetForemost Backend Test

## Env Variables

Estas variables son las necesesarias para iniciar el servidor

```
//Server port
PORT=
//Database host
DB_HOST=
//Database user
DB_USER=
//Database port
DB_PORT=
//Database password
DB_PASSWORD=
//Database name
DB=
```

## Scripts

### Start development server

```bash
$ yarn dev
```

### Start production server

```bash
$ yarn start
```
